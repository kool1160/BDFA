# UI Guidelines

## Design language

Apple-inspired, not Apple-copied.

## Rules

- Mobile-first layout
- Large readable numbers
- Soft cards
- Dark mode first
- Minimal tabs
- No clutter
- No dense finance tables on the home screen
- Every screen should answer one clear question

## Breakpoints

- iPhone: single-column cards
- iPad portrait: larger cards, tighter spacing
- iPad landscape: two-column dashboard/panels
- Desktop later: three-column command center

## Tone

Plain English. No finance jargon unless the user asks for it.

# BDFA Product Principles

This document defines durable product guidance for BDFA feature planning, UI decisions, AI behavior, copy, recommendations, and future implementation work.

## Financial Truth Over Financial Sales

BDFA exists to give users the most honest, complete, and actionable understanding of their financial life.

BDFA must work for the user.

Not advertisers.

Not banks.

Not investment companies.

Not affiliates.

Do not add features, copy, recommendations, UI flows, or AI behavior that:

- push financial products
- manipulate users
- hide tradeoffs
- prioritize monetization over clarity
- recommend products because of sponsorships
- make financial decisions without user approval

Prioritize:

- financial clarity
- user trust
- accurate source data
- transparent calculations
- helpful explanations
- reduced financial stress
- better decision-making
- honest “what should I do next?” guidance

Every feature should pass this test:

Does this help the user better understand their finances or make a better financial decision?

If not, reconsider it.

## Core Product Rule

Every BDFA feature must help users better understand their finances or make better financial decisions.

If a feature does not improve financial clarity, user trust, planning, safety, or decision-making, it should be reconsidered.

## AI Assistant Rule

BDFA's future AI assistant should behave like an honest financial guide.

The AI should:

- explain financial situations
- identify risks
- identify opportunities
- run what-if simulations
- forecast outcomes
- help users build better financial habits

The AI must not:

- push sponsored products
- manipulate users
- make decisions without approval
- hide uncertainty
- pretend derived values are source truth

## Monthly Flow Rule

Monthly Flow remains BDFA's flagship planning experience.

Traditional budgeting is secondary.

Monthly Flow should help users quickly understand:

- incoming income
- upcoming bills
- cash flow
- projected balances
- Safe to Spend
- Safe to Invest
- future outlook
